# Frontend Conventions

Applies to `packages/ui/`.

## Tooling

| purpose | tool |
|---------|------|
| package manager | `pnpm` |
| types | `tsc --noEmit` |

## Key Commands

```bash
pnpm ui:dev           # dev server
pnpm ui:build         # production build
pnpm ui:preview       # preview production build
```
